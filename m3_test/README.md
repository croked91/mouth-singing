# M3 Test — Эксперименты по улучшению слоговой разметки

Два изолированных эксперимента для сравнения подходов к разметке слогов
при наличии текста песни, но без LRC-таймингов.

## Планы

1. [Подготовка тестовых данных](./PLAN_PREPARE_DATA.md) — выполнить первым
2. [Вариант 2: WhisperX ASR → сопоставление с текстом → force_align](./PLAN_VARIANT2.md)
3. [Вариант 3: Sonoix + LLM-коррекция текста](./PLAN_VARIANT3.md)

## Тестовые данные

Для обоих экспериментов используется одна и та же тестовая выборка.
Для каждого трека заготавливаем:
  - MP3 файл (оригинал) — предоставляет пользователь
  - Вокал (результат UVR-сепарации) — генерируется скриптом подготовки
  - Чистый текст песни (из lyrics_text в БД)
  - Эталонная разметка (syllable_timings из БД — результат LRC + force_align при бутстрапе)
